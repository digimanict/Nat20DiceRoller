<H2>How It's Made</H2>
C#, XAML, MAUI

I wanted to compaire the experience of developing for Android devices with C# using MAUI in Visual Stuio to the traditional experience with Kotlin and Android Development Studio. I was surprised to discover that it is a viable way of developing cross platform apps quickly and effeicnetly. <br>

<H2>Conclusion</H2>
While Kotlin and Android Studio are meant for android development exclusively, Visual Studio once again proves to be a versatile workhorse. The large helpful community that surrounds Visual Studio, and the ability to deploy to multiple platforms from a single project, make MAUI a viable option for App Development.
